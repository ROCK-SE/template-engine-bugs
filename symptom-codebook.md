# Symptoms in Template Engine Application Bugs

## Configuration Error (23)
This category
- Template Not Found (17): The template engine fails to locate the template file.
- Dependency Error (4): The template engine and related dependencies are not installed, or are installed with wrong version, or are not correctly integrated.
- Encoding Error (2): The template engine's encoding setting can not recognize some characters in the template, resulting in garbled characters or encoding errors.
